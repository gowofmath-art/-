# 📐 Chapter 2.5 其他基本的數學知識

本篇章研讀米田優俊《演算法×數學》第 2.5 節，本章節列出前面沒提到的一些基本數學知識。  
但基本上對我來說都非常親切，算是給非數學科系的人來讀的。

但還是稍微總結一下提到的關鍵知識點：
1. 質數
2. 最大公因數、最小公倍數
3. 階乘
4. 數列的基礎
5. 集合的基礎  (有意思的是有提到空集合、屬於、交集、聯集、子集合等重要概念)
6. 必要條件和充分條件
7. 絕對誤差和相對誤差
8. 閉區間、半開區間、開區間
9. 求和符號sigma
10. 和的公式 (有提到：等差數列、平方和的公式)

對我來說全都是非常親切地數學語言，這些每個專有名詞，要我隨時講與寫出來都沒問題！

重要的是，本書前兩章都在學習基本的數學知識與概念，頂多只有簡單的程式，之後第3章才是開始學習演算法！
