3.5.2 蒙地卡羅法是什麼？  
蒙地卡羅法是使用隨機數的演算法，核心概念為：  
在推測某事的成功率時，進行n次隨機測試，其中m次成功。將理論上的成功率視為近似 $\frac{m}{n}$。  

3.5.3 應用例：圓周率 $\pi$ 的計算
接著，介紹「計算圓周率 $\pi$ 近似值的方法」作為蒙地卡羅法的應用例之一。可透過以下步驟計算 $\pi$ 的近似值：  
*   **步驟 1**：在一邊為 1cm 的正方形中隨機放置 $n$ 個點  
*   **步驟 2**：將進入以左下角為中心半徑 1cm 的圓內的點的個數設為 m  
*   **步驟 3**：此時，將 $\frac{4m}{n}$ 設為圓周率 $\pi$ 的近似值  


3.5.4 理論驗證之前(1)：平均、標準差  
在進行蒙地卡羅法的理論驗證之前，先整理一下基本的統計知識。介紹以下 2 個有名的數值，是用來表示資料或機率分布的大致特徵。  

*   **平均值 $\mu$**：資料的平均值  
*   **標準差 $\sigma$（Sigma）**：資料的分散狀況  

$N$ 個資料 $x_1, x_2, \dots, x_N$ 的平均值 $\mu$ 和標準差 $\sigma$ 被定義如下式。  

$$\mu = \frac{x_1 + x_2 + \dots + x_N}{N}$$  

$$\sigma = \sqrt{\frac{(x_1 - \mu)^2 + (x_2 - \mu)^2 + \dots + (x_N - \mu)^2}{N}}$$  

3.5.5 理論驗證之前(2)：常態分佈是什麼？  
介紹常態分布：
它會呈現一個以平均值為中心、左右完全對稱的「鐘型曲線」
常態分布的機率密度函數（PDF）常態分布的形狀完全由兩個參數決定：平均值 $\mu$（決定中心位置）與標準差 $\sigma$（決定分散程度）。  

其數學定義如下式：
$$f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{(x - \mu)^2}{2\sigma^2}}$$  

$x$：隨機變數。
$\mu$：分佈的平均值（對稱軸所在的中心點）。  
$\sigma$：標準差（控制鐘形曲線的寬度與胖瘦）。  
$\pi$：圓周率。  
$e$：自然對數的底數（歐拉數）。 

著名的「68-95-99.7 法則」（又稱常態分布經驗法則）：
在標準常態分布下，資料落再特定範圍內的機率具有以下幾何特徵：
約有 68% 的資料會落在距離平均值 1 個標準差的範圍內（$\mu \pm 1\sigma$）。  
約有 95% 的資料會落在距離平均值 2 個標準差的範圍內（$\mu \pm 2\sigma$）。  
約有 99.7% 的資料會落在距離平均值 3 個標準差的範圍內（$\mu \pm 3\sigma$）。  

3.5.6 從理論上驗證蒙地卡羅法  
如果n夠大，進行n次成功機率為p的測試時，n次中成功的比例可以近似為平均 $\mu = p$，標準差 $\sigma = \sqrt{\frac{p(1-p)}{n}}$ 的常態分布。

以下為個人補充，非書中上的知識點。  
當我們重覆進行 $n$ 次獨立試驗時，成功的**總次數** $X$ 服從二項分布 $X \sim B(n, p)$，此時總次數的：
*   平均值 $E(X) = np$
*   變異數 $Var(X) = np(1-p)$

然而，在實際應用中（例如估算圓周率 $\pi$），我們關注的是**「成功的比例（$\hat{p} = \frac{X}{n}$）」**。根據隨機變數的性質，將總次數除以 $n$ 後，成功比例的統計特徵會轉變為：

*   **平均值 $\mu$**：
    $$\mu = E\left(\frac{X}{n}\right) = \frac{np}{n} = p$$
*   **變異數 $Var(\hat{p})$**：
    $$Var\left(\frac{X}{n}\right) = \frac{1}{n^2} Var(X) = \frac{np(1-p)}{n^2} = \frac{p(1-p)}{n}$$
*   **標準差 $\sigma$**（變異數開根號）：
    $$\sigma = \sqrt{\frac{p(1-p)}{n}}$$

#### 🔍 蒙地卡羅法的核心結論

根據**中央極限定理（Central Limit Theorem）**，當試驗次數 $n$ 趨近於無窮大時，成功比例 $\hat{p}$ 的抽樣分佈會趨近於常態分布：

$$\hat{p} \sim N\left(p, \frac{p(1-p)}{n}\right)$$

從標準差 $\sigma = \sqrt{\frac{p(1-p)}{n}}$ 的公式中可以看出，誤差（標準差）與 **$\sqrt{n}$ 成反比**。這從理論上嚴格證明了：**只要我們大幅增加測試次數 $n$，標準差（估算誤差）就會逐漸縮小，讓計算出的近似值越來越精準。**
